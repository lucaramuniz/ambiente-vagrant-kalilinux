# Ambiente vagrant kalilinux
Ambiente default do kali para aplicações e testes rápidos

## Configurações Default
- **Provedor:** VirtualBox
- **Sistema Operacional:** Kali Linux
- **Versão:** Rolling/Lastest
- **CPU:** 1
- **Ram:** 2048Mb (~2Gb)
- **Rede:** Privada
- **Modo de Rede:** DHCP
- **Faixa de IP:** NULL
